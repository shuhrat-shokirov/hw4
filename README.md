# Домашнее задание #4: Форматирование цен

## 📋 Описание задачи

В этом задании вам нужно исправить функцию форматирования цен в консольном приложении для интернет-магазина Alifshop. Приложение принимает данные о товаре и выводит красиво отформатированную карточку товара с расчетом рассрочки.

### Что уже реализовано

✅ Консольное приложение для ввода данных о товаре
✅ Расчет ежемесячного платежа при рассрочке на 12 месяцев
✅ Хранение цены в тийинах (правильный подход для работы с деньгами)
✅ Базовые тесты для проверки функциональности

### В чем проблема?

❌ Функция `Converter` выводит цены как есть (в тийинах), без форматирования
❌ Нет разделения тысяч пробелами
❌ Тийины всегда показываются, даже когда равны 00

## 🎯 Что нужно сделать

1. **Форкнуть репозиторий**: https://github.com/shuhrat-shokirov/hw4

2. **Исправить функцию форматирования** в файле `internal/product/printer.go`:
    - Конвертировать тийины в сумы (делить на 100)
    - Разделять целую часть пробелами по разрядам: `12990000` → `12 990 000`
    - Показывать тийины только если они есть:
        - Если тийины = 00: `12 990 000 сум`
        - Если тийины ≠ 00: `12 990 123.99 сум`
    - Применить форматирование к цене и ежемесячному платежу

## 📝 Примеры

### Пример 1: Цена без тийинов
**Вход:** `Price = 12_990_000_00` (12 990 000 сумов в тийинах)

**Ожидаемый вывод:**

```
===== Alifshop =====
Товар:    ABC
Бренд:    a
Цена:     12 990 123.99 сум
В наличии: true
Рассрочка: 12 мес → 1 082 510.33 сум/мес
====================
```

## 💡 Подсказки

1. **Разделение тысяч**: Идите с конца строки и вставляйте пробел каждые 3 символа
2. **Работа с тийинами**: Используйте операции деления и остатка от деления на 100
3. **Форматирование**: Для тийинов используйте формат с двумя знаками после запятой (например, `.99`, а не `.9`)
4. **Отрицательные числа**: Можно не обрабатывать специально - тесты их уже корректно обрабатывают

## ❓ Часто задаваемые вопросы

**Q: Нужно ли обрабатывать отрицательные цены?**
A: В тестах они есть только для функции Calculate. Специальная обработка не требуется.

**Q: Как правильно форматировать тийины?**
A: Всегда с двумя знаками после запятой: `.01`, `.50`, `.99`

**Q: Что делать если тийины равны 00?**
A: Не показывать их вообще: `1 000 сум`, а не `1 000.00 сум`
